# i3 Window Manager

Tiling window manager configuration with vim-style navigation and extensive keybindings.

## Config Location

`i3/config`

## Key Configuration

### Modifier Key

```
set $mod Mod1  # Alt key
```

### Font

```
font pango:JetBrainsMono Nerd Font 14
```

### Focus Behavior

```
focus_on_window_activation none  # Prevent focus stealing
```

## Keybindings

### Core

| Key | Action |
|-----|--------|
| `Mod+Return` | Open terminal (Alacritty) |
| `Mod+d` | App launcher (Rofi) |
| `Mod+Shift+q` | Kill focused window |
| `Mod+Shift+r` | Reload i3 config |
| `Mod+Shift+e` | Exit i3 |

### Window Navigation

| Key | Action |
|-----|--------|
| `Mod+h` | Focus left |
| `Mod+j` | Focus down |
| `Mod+k` | Focus up |
| `Mod+l` | Focus right |
| Arrow keys | Same with arrows |

### Window Movement

| Key | Action |
|-----|--------|
| `Mod+Shift+h` | Move left |
| `Mod+Shift+j` | Move down |
| `Mod+Shift+k` | Move up |
| `Mod+Shift+l` | Move right |

### Workspaces

| Key | Action |
|-----|--------|
| `Ctrl+j` | Workspace 1 |
| `Ctrl+k` | Workspace 2 |
| `Ctrl+l` | Workspace 3 |
| `Ctrl+;` | Workspace 4 |
| `Ctrl+Shift+<key>` | Move to workspace |

### Layout

| Key | Action |
|-----|--------|
| `Mod+s` | Stack layout |
| `Mod+w` | Tab layout |
| `Mod+e` | Toggle split |
| `Mod+f` | Fullscreen |
| `Mod+Shift+space` | Toggle floating |

### Splitting

| Key | Action |
|-----|--------|
| `Mod+h` | Split horizontal |
| `Mod+v` | Split vertical |

### Resize Mode

Enter with `Mod+r`:

| Key | Action |
|-----|--------|
| `h` | Shrink width |
| `l` | Grow width |
| `k` | Shrink height |
| `j` | Grow height |
| `Escape` | Exit mode |

### System Toggles

| Key | Action |
|-----|--------|
| `Mod+Shift+t` | Toggle theme |
| `Mod+Shift+n` | Toggle focus mode |
| `Mod+Shift+b` | Toggle polybar |
| `Mod+Shift+c` | Toggle screen recording |
| `Mod+Shift+a` | Toggle lid suspend |

### Voice Dictation

| Key | Action |
|-----|--------|
| `Mod+v` | Start/stop recording |
| `Mod+Shift+v` | Cancel recording |

### Media Keys

| Key | Action |
|-----|--------|
| `XF86AudioRaiseVolume` | Volume up |
| `XF86AudioLowerVolume` | Volume down |
| `XF86AudioMute` | Mute toggle |
| `XF86MonBrightnessUp` | Brightness up |
| `XF86MonBrightnessDown` | Brightness down |
| `XF86AudioPlay` | Play/pause Spotify |
| `XF86AudioNext` | Next track |
| `XF86AudioPrev` | Previous track |

### Screenshots

| Key | Action |
|-----|--------|
| `Print` | Screenshot (Flameshot) |

## Window Rules

### Floating Windows

```
for_window [class="Spotify"] floating enable
for_window [window_role="pop-up"] floating enable
for_window [window_role="dialog"] floating enable
```

### Workspace Assignments

```
assign [class="Brave-browser"] $ws4
assign [class="firefox"] $ws4
```

## Startup Applications

```
exec_always --no-startup-id polybar
exec --no-startup-id dunst
exec --no-startup-id lid-suspend-sync
```

## Colors

Colors managed by tinty. Imported from generated file:

```
include ~/.config/i3/colors
```

See [features/theming.md](../features/theming.md).

## Gaps (if enabled)

```
gaps inner 10
gaps outer 5
```

## Border

```
default_border pixel 2
```

## Adding Custom Bindings

Add to `i3/config`:

```
bindsym $mod+x exec my-command
```

Then reload: `Mod+Shift+r`
