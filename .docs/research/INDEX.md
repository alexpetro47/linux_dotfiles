# Research Documentation Index

Quick reference guide for research documents in this directory.

## Documents

### bitwarden-cli-secrets-management.md
Comprehensive research on Bitwarden CLI (bw) and Secrets Manager (bws) for managing API keys and environment variables across projects. Covers installation, authentication, vault/project organization, integration patterns (local dev, CI/CD), security considerations, and recommended workflows. Includes comparison matrix (bw vs bws), gotchas/anti-patterns, and practical code examples.

### respondus-lockdown-browser-vm-detection.md
Research on Respondus Lockdown Browser detection mechanisms and strategies for running it in virtual machines. Documents detection methods, circumvention techniques, and VM configuration approaches.

**How to add new documents**: Create your `.md` file, then add a brief 1-2 sentence description here pointing to what's inside.
